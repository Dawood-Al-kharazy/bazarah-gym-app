# Bazarah Gym Management System (جيم سكن بازرعة) 

A premium, fully offline-capable mobile application built with **Flutter** to streamline gym management operations. Designed with a sleek Dark-Themed Glassmorphism UI, this app offers a comprehensive suite of tools for managing members, tracking capacity, and generating Arabic PDF reports.

## ✨ Features

- **Offline-First Architecture**: Built on top of `SharedPreferences` to ensure blazing-fast performance and data security without requiring an internet connection.
- **Smart Member Management**: Quickly add new members, set start dates, and select training periods with a few taps.
- **One-Tap Renewals**: Edit member details or renew subscriptions (15 or 30 days) instantly from a unified dialog.
- **Duplicate Protection**: Real-time validation to prevent accidental duplicate entries.
- **Interactive Dashboard**: Track capacity across 6 daily periods with color-coded status indicators (Active, Expiring Today, Expired).
- **Arabic PDF Reports**: Custom implementation for generating and exporting printable gym reports with perfect RTL and custom font (Tajawal) support.
- **Data Backup**: Simple backup capabilities to safeguard gym data.

## 🎨 UI & Design

The application features a modern **Dark-Mode Glassmorphism** design, prioritizing user experience and aesthetic appeal:
- Custom **Tajawal** typography for elegant Arabic rendering.
- Smooth micro-animations for interactive elements.
- Clean, intuitive navigation using a bottom tab bar.

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Dawood-Al-kharazy/Bazarah-Gym-App.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Bazarah-Gym-App
   ```
3. Get the dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

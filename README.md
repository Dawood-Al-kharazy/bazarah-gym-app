## ✨ Features | المميزات

- 🔒 **Offline-First Architecture**: Built on top of local storage for blazing-fast performance and robust data security without requiring an internet connection.
- 👥 **Smart Member Management**: Quickly add new members, set exact start dates, and select training periods with a few taps.
- ⚡ **One-Tap Renewals**: Edit member details or renew subscriptions (15 or 30 days) instantly from a unified, intuitive dialog.
- 🛡️ **Duplicate Protection**: Real-time validation to prevent accidental duplicate entries.
- 📊 **Interactive Dashboard**: Track capacity across 6 daily periods with color-coded status indicators (Active, Expiring Today, Expired).
- 📄 **Arabic PDF Reports**: Generate and export printable gym reports with perfect RTL support and custom typography (Tajawal).
- 💾 **Data Backup**: Simple backup and restore capabilities to safeguard gym data.

## 🎨 UI & Design | التصميم وواجهة المستخدم

The application features a modern **Dark-Mode Glassmorphism** design, prioritizing user experience and aesthetic appeal:
- Custom **Tajawal** typography for elegant Arabic rendering.
- Smooth micro-animations for interactive elements.
- Smart keyboard management and dynamic layout adjustments.
- Clean, intuitive navigation using a customized bottom tab bar.

## 🛠️ Built With | التقنيات المستخدمة

- **[Dart](https://dart.dev/)** - The programming language.

## 🚀 Getting Started | التشغيل المحلي

To run this project locally / لتشغيل المشروع على جهازك:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dawood-Al-kharazy/Bazarah-Gym.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd Bazarah-Gym
   ```
3. **Get the dependencies:**
   ```bash
   flutter pub get
   ```
4. **Run the app:**
   ```bash
   flutter run
   ```
